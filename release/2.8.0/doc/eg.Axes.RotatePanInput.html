Class: RotatePanInput

eg.Axes. RotatePanInput

new eg.Axes.RotatePanInput(element, options)

RotatePanInput.js, line 17

A module that passes the angle moved by touch to Axes and uses one axis of rotation.
Details

터치에 의해 움직인 각도를 Axes 에 전달하며 1개의 회전축만 사용한다.
상세내용

  • element
    Type: HTMLElement | String | jQuery

    An element to use the eg.Axes.RotatePanInput module

    eg.Axes.RotatePanInput 모듈을 사용할 엘리먼트

  • options optional
    Type: PanInputOption

    The option object of the eg.Axes.PanInput module

    eg.Axes.PanInput 모듈의 옵션 객체

Example
const input = new eg.Axes.RotatePanInput("#area");


var axes = new eg.Axes({

// property name('angle') could be anything you want (eg. x, y, z...)

angle: {

range: [-180, 180] // from -180deg to 180deg

}

});


axes.connect("angle", input)
