```ts
event release
```

<div className="bulma-tags">

</div>

This event is fired when a user release an element on the screen of the device.

**Type**: object

|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|depaPos|Object&lt;string, number&gt;|손을 뗐을 때의 좌표 |
|destPos|Object&lt;string, number&gt;|손을 뗀 뒤에 이동할 좌표|
|delta|Object&lt;string, number&gt;|좌표의 변화량|
|inputEvent|Object|inputType으로 부터 받은 이벤트 객체|
|input|Object|이벤트가 발생한 inputType 인스턴스|
|setTo|[setTo](setTo)|이벤트 이후 이동할 애니메이션 좌표를 지정한다|
|isTrusted|Boolean|사용자의 액션에 의해 이벤트가 발생하였으면 true, 스크립트나 API호출에 의해 발생하였을 경우에는 false를 반환한다.|

```js
const axes = new eg.Axes({
  "x": {
     range: [0, 100]
  },
  "zoom": {
     range: [50, 30]
  }
}).on("release", function(event) {
  // event.depaPos
  // event.destPos
  // event.delta
  // event.input
  // event.inputEvent
  // event.setTo
  // event.isTrusted

  // if you want to change the animation coordinates to move after the 'release' event.
  event.setTo({x: 10}, 2000);
});
```
