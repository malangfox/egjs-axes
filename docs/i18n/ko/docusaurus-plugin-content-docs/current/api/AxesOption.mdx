The option object of the eg.Axes module

**Type**: Object

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|easing|function|✔️|easing.easeOutCubic|애니메이션에 적용할 easing 함수|
|maximumDuration|Number|✔️|Infinity|가속도에 의해 애니메이션이 동작할 때의 최대 좌표 이동 시간|
|minimumDuration|Number|✔️|0|가속도에 의해 애니메이션이 동작할 때의 최소 좌표 이동 시간|
|deceleration|Number|✔️|0.0006|사용자의 동작으로 가속도가 적용된 애니메이션의 감속도. 값이 높을수록 애니메이션 실행 시간이 짧아진다|
|interruptable|Boolean|✔️|true|진행 중인 애니메이션 중지 가능 여부.<br />- true: 사용자의 동작이나 API로 애니메이션을 중지할 수 있다.<br />- false: 애니메이션이 진행 중일 때는 사용자의 동작이나 API가 적용되지 않는다|
|round|Number|✔️||반올림 단위. 예를 들어 0.1 은 소숫점 0.1 까지 반올림(6.1234 => 6.1), 5 는 5 단위로 반올림(93 => 95).<br />[상세내용](https://github.com/naver/egjs-axes/wiki/round-option)|
