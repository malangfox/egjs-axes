The Axis information. The key of the axis specifies the name to use as the logical virtual coordinate system.

**Type**: Object

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|range|Array&lt;Number&gt;|✔️||좌표 범위|
|range[0]|Number|✔️|0|최소 좌표|
|range[1]|Number|✔️|0|최대 좌표|
|bounce|Array&lt;Number&gt;|✔️||바운스 영역의 크기. 사용자의 동작에 따라 좌표가 좌표 영역을 넘어 바운스 영역의 크기만큼 더 이동할 수 있다. 사용자가 끌어다 놓는 동작을 했을 때 좌표가 바운스 영역에 있으면, 바운스 효과가 적용된 좌표가 다시 좌표 영역 안으로 들어온다|
|bounce[0]|Number|✔️|0|최소 좌표 바운스 영역의 크기|
|bounce[1]|Number|✔️|0|최대 좌표 바운스 영역의 크기|
|circular|Array&lt;Boolean&gt;|✔️||순환 여부. 'true'로 설정한 방향의 좌표 영역 밖으로 엘리먼트가 이동하면 반대 방향에서 엘리먼트가 나타난다|
|circular[0]|Boolean|✔️|false|최소 좌표 방향의 순환 여부|
|circular[1]|Boolean|✔️|false|최대 좌표 방향의 순환 여부|
