event animationEnd

This event is fired when animation ends.

**Type**: object

|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|isTrusted|Boolean|사용자의 액션에 의해 이벤트가 발생하였으면 true, 스크립트나 API호출에 의해 발생하였을 경우에는 false를 반환한다.|

```js
const axes = new eg.Axes({
  "x": {
     range: [0, 100]
  },
  "zoom": {
     range: [50, 30]
  }
}).on("animationEnd", function(event) {
  // event.isTrusted
});
```
